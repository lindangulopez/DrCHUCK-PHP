If you want to write semantic code it is important that you understand these differences:

  • The i element was for italic text, now it is also for alternate text, such as foreign words, technical terms, or inline stage directions
  • So, for alternate accentuation: use the em, not the italic element.
  • As the b element is no longer used for bold text, now it is also used as a stylistic offset such as keywords in a document of product names like above.

If none of this is clear read the docs
